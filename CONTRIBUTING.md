# Contribuindo

Contribuições são bem-vindas para correções, melhorias, refatorações e sugestões relacionadas ao projeto.

Antes de abrir um pull request, considere:

- Manter o objetivo original do projeto
- Evitar mudanças grandes sem discussão prévia em uma issue
- Seguir o padrão de arquitetura existente
- Incluir testes quando a alteração afetar comportamento
- Manter o código simples e legível

Ao enviar uma contribuição, você concorda que ela poderá ser incorporada ao projeto sob os mesmos termos definidos na licença do repositório.
